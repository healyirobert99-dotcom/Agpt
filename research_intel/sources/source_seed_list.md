# AlphaGPT 信息收集来源种子清单

本清单只登记公开资料来源，用于后续 Firecrawl 或降级采集。不得抓取登录、付费、受限或 robots/条款禁止的内容。

## 优先来源

1. 聚宽社区公开文章与策略讨论
   - 目标内容：多因子选股、小市值、ETF 轮动、动量反转、调仓与风控。
   - 采集方式：Firecrawl search 优先；只采集公开可访问页面。

2. 米筐 / RiceQuant 公开文章与研究内容
   - 目标内容：因子研究、组合构建、回测假设、交易成本说明。
   - 采集方式：Firecrawl search 或公开页面 scrape。

3. BigQuant 公开策略和因子文章
   - 目标内容：多因子选股、机器学习选股、量价因子、基本面因子。
   - 采集方式：Firecrawl search 或公开页面 scrape。

4. GitHub 公开量化项目
   - 目标内容：A 股多因子、选股、轮动、指数增强、开源回测框架。
   - 代表来源：Microsoft Qlib、公开 A 股因子/策略项目。

5. 公开研报、论文和博客
   - 目标内容：Smart Beta、因子投资、低波动、质量、价值、动量、反转、流动性。
   - 注意：只记录摘要和因子逻辑，不复制完整研报或完整策略代码。

## 第一批代表性公开样本

- Firecrawl GitHub / 官方文档：用于部署和采集能力确认。
- Microsoft Qlib GitHub：用于学习公开量化工作流、Alpha158/Alpha360 等特征工程先例。
- Qlib 论文：用于记录“从研究到回测到交易执行”的完整链条。
- Smart Beta 公开条目：用于因子投资基础分类。
- Stockformer 公开论文摘要：用于量价、多频、图关系类因子方向。
- 公开多因子市场中性论文摘要：用于组合构建、风险平价、风险控制参考。
- 公开 A 股多智能体基本面投资论文摘要：用于宏观、行业、公司基本面与风险控制层次。

## 边界

- 本阶段不运行 AlphaGPT 回测。
- 本阶段不启动第二批搜索。
- 本阶段不修改 AlphaGPT 因子生成器、筛选阈值、评级规则或 Phase 2 逻辑。
- 所有库记录仅作为后续研究的种子先验，尚未经过 AlphaGPT 本地验证，不得用于交易。
