# Research Boundary

This repository is for research coordination and safe public synchronization.

Allowed public contents:

- source code without secrets;
- tests;
- non-secret configuration files;
- lightweight research reports;
- public-source factor prior summaries;
- public-source trading-operation strategy summaries;
- Firecrawl wrapper code without API keys.

Excluded contents:

- local market databases;
- raw market data;
- raw crawl data;
- API keys;
- cookies;
- account credentials;
- broker information;
- full copyrighted reports;
- large run artifacts;
- real trading signals;
- buy/sell recommendations.

All factor and strategy records are unvalidated seed candidates unless a later AlphaGPT local backtest explicitly validates them.
